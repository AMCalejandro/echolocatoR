echolocatoR: plotting vignette

Brian M. Schilder

Most Recent Update:

library(echolocatoR)

Plotting only

echolocatoR contains various functions that can be used separately
from the comprehensive finemap_loci() pipeline.

Generate a multiview plot using GGBIO.plot(). * You can mix and match different tracks and annotations using the different arguments (see ?GGBIO.plot for details).
* Zoom in/out by adjusting the plot.window size (in units of basepairs).
The plot is centered on the lead/index SNP.
* GGBIO.plot() returns a ggbio track object. + Further modify this object using ggplot2 functions like + theme(). + The modifications will be applied to all tracks at once.
* Save a high-resolution .png of the plot by setting save_plot=T
+ Further increase resolution by adjusting the dpi argument (default=400). + Adjust the height and width of the saved plot using these respective arguments. + The plot will be automatically saved in the locus-specific directory as:
*multiview__<plot.window>.png*.
